A/A/B Test — Font Redesign
Funnel Analysis

Statistical validation of whether a proposed UI font redesign significantly impacted user behavior across each stage of a mobile app's purchase funnel.

Python Pandas / NumPy SciPy Matplotlib Plotly A/A/B Testing Funnel Analysis
View on GitHub →
01 — Problem

Context & Objective

A product team proposed updating the font across their mobile application, hypothesizing it would improve readability and, in turn, drive higher conversion rates through the purchase funnel. Before committing to the redesign, an A/A/B experiment was designed to rigorously evaluate the change.

The experiment involved three user groups: two control groups (A and A₂, with identical UI) used to validate that the testing methodology itself was sound, and one treatment group (B, with the new font). With 7,551 total users distributed across the groups, the analysis measured conversion at five funnel stages.

Funnel Stages

The five stages tracked were: Main Screen → Offers Screen → Cart → Payment Successful → Tutorial. Each stage was tested independently using two-sample t-tests, with a significance level (α) set at 0.1.

03 — Statistical Analysis

A/A/B Comparison by Stage

Each funnel stage was independently tested for statistical differences between the three groups. The null hypothesis (H₀) in each case was that conversion rates between any two groups are equal, with the alternative (H₁) being that they differ.

Hypothesis Framework

H₀: Conversion rates across groups are equal — the font change has no effect.
H₁: Conversion rates differ between groups — the font change had an impact.

Across all five stages and all three group comparisons (A vs A₂, A vs B, A₂ vs B), the resulting p-values were consistently well above the significance threshold of α = 0.1. In every case, H₀ was not rejected, indicating no statistically significant difference attributable to the font redesign.

04 — Conclusion

Key Findings

The A/A/B experiment conclusively showed that the proposed font redesign did not significantly alter user behavior at any stage of the purchase funnel. Across all five stages and all three group comparisons, the p-values remained well above the significance threshold, meaning the null hypothesis was consistently retained.

The significance level was set at α = 0.1 — had it been set lower, the outcome would remain unchanged, as none of the p-values were close to the boundary. The variability in user counts between groups was minimal, further confirming the experiment's integrity.

Business recommendation: Since the font redesign carries no measurable risk to conversion, the team can implement it for aesthetic or accessibility reasons without concern. However, to meaningfully improve conversion metrics, optimization efforts should be focused on the Tutorial stage, which exhibits the largest drop-off in the entire funnel (only 23% retention from Cart).
